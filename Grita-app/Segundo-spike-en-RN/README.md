### **Manual de usuario**

Paso 1: Clonar repositorio de github en Documentos.
Paso 2: ingresar a la carpeta llamada Segundo-spike-en-RN.
Paso 3: Requisitos: verificar si esta el bun istalado codigo: ( bun--version ).En caso de que no tengas el bun instalar ( bun install ). Tener instalado expo go en tu celular (Android/iOS).
Paso 4: Una vez instalado correr el servidor dentro de la carpeta "Segundo-spike-en-RN" con: ( bun start (Alternativas en caso de que no funcione) bunx expo start / npm run start / npx expo start ) Se abrirá Expo DevTools y verás un QR en la terminal.
Paso 5: Escanear el QR con la aplicacion Expo go de tu celular y veras la pagina.
Paso 6: Cómo usar:
Escribí un producto y tocá **Agregar** o **Enter**.
Tocá un ítem para **marcar/desmarcar** (hecho).
**Dejá apretado** un ítem para **eliminarlo**.
