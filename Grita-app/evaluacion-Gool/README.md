Manual de Usuario: "El Grito Sagrado"

Obra de literatura digital interactiva inspirada en "Grita" de José Aburto. Esta aplicación utiliza el esfuerzo físico del usuario (el grito) para invocar la memoria colectiva del fútbol argentino.

1. Requisitos Técnicos

   Entorno: Bun instalado.

   Hardware: Dispositivo móvil físico con micrófono funcional.

   Software: Aplicación Expo Go instalada.

2. Instalación

   Navegar a la carpeta del proyecto.

   Instalar dependencias con Bun:

   bun install

   Asegurarse de haber colocado los archivos maradona.mp3 y messi.mp3 en la carpeta assets/audio/.

3. Ejecución

   Iniciar el servidor de desarrollo:

   bunx expo start

   Escanear el código QR con la aplicación Expo Go.

   Otorgar los permisos de micrófono solicitados.

4. Modo de Uso

   Colocar el dispositivo frente a la boca.

   Iniciar un grito de gol potente y sostenido.

   Interacción: El relato aparecerá en pantalla y el audio original comenzará a reproducirse solo mientras el volumen del grito sea elevado.

   Finalización: Si el usuario hace silencio, el relato se detiene y la pantalla se oscurece, exigiendo un nuevo esfuerzo físico para continuar.
